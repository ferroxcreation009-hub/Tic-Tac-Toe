# Tic Tac Toe Game 🎮

A classic two-player Tic Tac Toe game built from scratch 
using pure HTML, CSS and JavaScript.

## Features
- Two player gameplay (O vs X)
- Win detection for all 8 patterns
- Draw detection
- New Game & Reset button
- Animated background theme

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Play
1. Player O goes first
2. Players take turns clicking boxes
3. First to get 3 in a row wins
4. If all boxes filled with no winner = Draw
